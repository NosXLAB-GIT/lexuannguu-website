# 🚧 Lexuannguu Wiki

> Website đang xây dựng với phong cách hiện đại, dark tech, UI hướng trải nghiệm.

---

## 🌌 Giới thiệu

**Lexuannguu Wiki** là dự án website dạng wiki, hướng tới việc tổng hợp nội dung, tài liệu và kiến thức theo cách trực quan, dễ dùng và hiện đại.

Hiện tại dự án đang trong giai đoạn xây dựng giao diện (UI/UX).

---

## 🎨 Demo hiện tại

- Dark Mode
- Orbit Animation (ảnh xoay quanh trung tâm)
- Glass UI + Glow Effect
- Loading Animation
